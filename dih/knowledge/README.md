# Knowledge (what the editor learns from)

Drop anything you want the script generator to learn Dog is Human's voice and
facts from. Everything here is read and fed to Gemini as reference before it
writes each script.

Good things to add:
- Brand voice / tone guidelines
- Product descriptions, ingredient lists, claims you're allowed to make
- Past winning scripts / ad copy
- Do-not-say list (banned claims, compliance language)
- Customer language / reviews to echo

Supported: `.md .txt .pdf` (PDFs are text-extracted). Keep files focused; the
whole folder is loaded, so trim anything stale.

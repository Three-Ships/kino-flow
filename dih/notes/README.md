# Notes (the brief for each video)

Each video is driven by one notes file. Two ways to create them:

1. **Paste in the UI** — the fastest path. Type/paste the DiH Slack note, give it
   a title, hit Add. It's saved here as a `.md` file.
2. **Drop a file here** — copy the Slack note into a new `.md` or `.txt` file.
   One file = one video. Name it something like `2026-07-21-itchy-skin-hook.md`.

## What to put in a note
Anything the client told you: the angle, the product to feature, must-say points,
the hook idea, the audience, the platform (TikTok vs Meta). Freeform is fine —
Gemini reads it as the brief. Example:

```
Platform: TikTok
Product: Itch relief kit
Angle: dog owner exhausted by 3am scratching, tried everything
Must say: vet-formulated, 30-day
Hook idea: "I almost gave up on my dog's skin..."
```

Files starting with `_` (like `_template.md`) and `README.md` are ignored by the
batch runner.
